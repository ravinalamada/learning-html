The description of the attribute rules

Attribute rules are the scripts that a user can define on dataset. They trigger automatically with incoming edits and can be used to constrain the attribute values allowed on fields or perform a calculation to derive a field's value.

These are some rules, which you should follow in order to gain a high quality of your code:

  • The attribute value needs quotes
  • The attributes are separated with either space or new line
  • The attributes name need to be lowercase
  • The element types have private attributes
  • Public attributes can go in any element like class or id
  • No quotes inside of double quotes
