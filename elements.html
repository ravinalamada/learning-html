Definition of html

Html is the standard markup language for documents designed to be displayed in a web browser.

What bits make up an element

Elements are the bits that make up web pages. It represents semantics, or meaning. For example, the title element represents the title of the document. Most HTML elements are written with a start tag (or opening tag) and an end tag (or closing tag), with content in between.
