The nesting rules of html

Nested rules are clever about handling selector lists (that is, comma-separated selectors). Each complex selector (the ones between the commas) is nested separately, and then they're combined back into a selector list.

These are the rules, which you need to follow :

  • All tags must be nested properly
  • If you insert other tags, close them in reverse order.
  • Indent the child elements
  • Every start tag needs to be ended.
  • You can't have end tag without start tag.
