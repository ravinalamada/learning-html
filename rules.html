List of html rules

There are many rules you need to follow when you will write html. It is not hard to remember, but you need to be more careful when you are writing it, and your work will be done perfectly.

These are the rules of writing html:

  • Validator: Validate the code
  • Structure: Try to keep the site structure simple.
  • Testing: Don't forget to test your work in a browser.
  • Indenting: Indent the child elements, which you use so that your work will be neat.
  • Proofread: Reread your work at least 3 times or ask your friend to read it in order to check spelling and grammar mistakes in it.
  • Closed tag: You can't have an end tag without a start tag.
  • Start tag: The start tags must be closed with end tags unless you use void element, which are the elements that don't contain any content.
  • Element rules: Use lowercase for the elements name.
  • The attribute rules: It is used to specify which rulings (between rows and columns) are to be shown in a table.
  • The nesting rules: Nest it properly. Make sure you end the element inside of the element.
