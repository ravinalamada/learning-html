List of html rules

There are many rules you need to follow when you write html. It is not hard to remember, but you need to be more careful when you write, and you work will be done perfectly.

These are the rules of writing html:

  • Validate the code
  • Try to keep the site structure as simple as possible
  • Include statement type (doctype) in order to avoid the quirks mode
  • Don't forget to test your work in a browser
  • Indent the child of the elements, which you use to make it neat
  • Check spelling mistakes of your work
  • Make sure you end the child elements when you nest it
  • Use lowercase for element name.
